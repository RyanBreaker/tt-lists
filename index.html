        Command Abilities Command Ability Details
        Mighty Destroyer You can use this command ability in your hero phase. The model that issues the command must be an IRONJAWZ model and the unit that receives the command must be an IRONJAWZ unit.

        - If the unit that received the command is within 3" of an enemy unit, you must make a pile-in move with each model in the unit.
